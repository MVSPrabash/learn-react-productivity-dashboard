# Productivity Dashboard

### React Learning Project

## Description

A simple React-based project built to learn and practice core React concepts such as components, state management, and hooks by creating a productivity dashboard.

---

## Features

* Dashboard with multiple widgets:

  * Clock
  * Notes
  * To-do list
* Modular component-based structure
* More widgets planned

---

## Tech Stack

* React
* JavaScript (ES6+)
* CSS
* Vite

---

## Installation

```bash
git clone https://github.com/MVSPrabash/learn-react-productivity-dashboard
cd learn-react-productivity-dashboard
npm install
npm run dev
```

---

## Usage

Open http://localhost:5173 in your browser.
